🎈Loon插件 JavDB-search.plugin

1️⃣提取Jellyfin/Emby媒体库视频下方MetaTube刮削器元数据链接的番号跳转到JavDB搜索。

2️⃣Google搜索番号跳转到JavDB搜索，前缀fh/jav 番号(中间有个空格，番号加不加连字符-均可），唤起位置不限于各浏览器默认搜索引擎，Siri建议。
